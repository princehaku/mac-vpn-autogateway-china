mac-vpn-autogateway-china
=========================


在你使用vpn的时候 自动进行服务端网络判断 

国内流量走原来网关 

国外的流量再走vpn


节省vpn资源  节省加载时间

使用方式

     git clone https://github.com/princehaku/mac-vpn-autogateway-china.git
     
     cd mac-vpn-autogateway-china
     
     sudo ./install.sh

然后 enjoy it!

mac同学一定记得勾上 选项里面的 通过VPN链接发送所有流量
